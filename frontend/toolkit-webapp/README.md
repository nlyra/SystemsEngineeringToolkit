### Front End

### Instructions:

# Installation
1. Download Node.js.
2. in cmd inside folder, run:
    ```npm install```
    ```npm install react-bootstrap bootstrap```
3. Create a JSON file called ```config.json``` inside src folder containing:
    {
        "server_url": "(( Server URL ))",
        "paths": {
            "login": "/v0/auth/login"
        }
    }

# Run
1. run in CMD ```npm start```.
